# DailyTodo

> 上班摸鱼做的小玩具,一个简单的flutter项目

## Features
- [x] Show daily todos in calendar mode
- [x] Add mood stickers for everyday
- [x] Store these todos and moods locally

## Run
```
// Run locally
flutter run

// Android package 
flutter build apk
```
